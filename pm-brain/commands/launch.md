# /launch — Launch Planning

Build a complete launch plan. Follows the growth skill's operating protocol (context loading, quality evaluation, context updates). Steps below define what to produce.

## Invocation

```
/launch AI-powered proposal writer for consulting firms
/launch New enterprise tier for our project management tool
```

## Scope Detection

- **Quick launch review** ("Are we ready to launch?"): Run readiness checklist only. ~500 words.
- **Feature launch** (known product, new feature): GTM + battlecard + readiness. ~1,000 words.
- **Full launch plan** (new product or `--deep`): All steps below. ~2,000 words.

## Workflow

### Step 1: Context + Gaps

Read all context files. Flag missing files with specific impact on launch planning. If `product-brief.md` missing → BLOCK. Others → WARNING.

### Step 2: Beachhead & ICP

Score 3-5 candidate segments on: burning pain, willingness to pay, winnability, referral potential. Select beachhead. Define ICP specific enough to find 10 real people.

### Step 3: GTM Strategy

Select motion (PLG/sales-led/community/hybrid) with rationale. Rank 2-4 channels by expected CAC. Craft positioning against actual competitors. Develop messaging by stakeholder.

### Step 4: Battlecard

For the primary competitor: comparison table, win themes, objection handling, landmines, pricing comparison. Use WebSearch for current data (or ask user if unavailable).

### Step 5: Growth Loops

Evaluate all four loop types (viral, content, paid, sales). Design primary loop with quantitative estimates: loop coefficient, cycle time, 6-month projection.

### Step 6: Launch Readiness

Assess 8 dimensions (Product, Marketing, Sales, Support, Engineering, Legal, Analytics, Ops). Score each. Flag must-haves. Define rollback triggers with specific thresholds. Recommend: launch / launch with mitigations / delay.

### Step 7: Compile

Single launch plan document: executive summary, beachhead, ICP, GTM, battlecard summary, growth loops, readiness matrix, timeline, success metrics (7/30/90 day), rollback triggers, risks.
